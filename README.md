# 📊 Student Performance Data Analysis

## 🔍 Overview
This project analyzes student performance using Python and SQL.

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- SQL

## 📈 Features
- Calculated average scores
- Identified top-performing students
- Analyzed gender-wise performance
- Visualized data using charts

## 📂 Files
- analysis.py → Python analysis script
- students.csv → Dataset
- analysis.sql → SQL queries

## 🚀 How to Run
1. Install dependencies:
   pip install pandas matplotlib

2. Run:
   python analysis.py

## 📊 Output
- Bar charts for average scores
- Gender-based comparisons
- Score distribution graphs
